About this project

The main purpose is to give people the opportunity to learn some things about sailing. We (Odhran and Tiernan) love to sail and we would like others to try it out too.

Most of this project was written in HTML5 using Bootstrap CSS to give some layout to the pages. Each lesson is a seperate HTML page, so that people can restart where they stopped. The answers to the quizs are checked with Javascript.
