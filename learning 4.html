About this project

The main purpose is to give people the opportunity to learn some things about sailing. We (Odhran and Tiernan) love to sail and we would like others to try it out too.

Most of this project was written in HTML5 using Bootstrap CSS to give some layout to the pages. Each lesson is a seperate HTML page, so that people can restart where they stopped. The answers to the quizzes are checked with Javascript.

Floating objects

Buoys

Now I think it's time we start learning about objects floating on the water.

The first object for this lesson is a buoy.

  1. A buoy is a floating circular ball on the water they are usualy used to mark a course but they can be used for other things.

  2. They are usually made in bright colours so that you can see them.

Here is a picture of a buoy!



Here are some bouys out of the water.

Buoys come in all sorts of shapes and sizes

In sailing terms, port means left and starboard means right. If someone is approaching you on the water, if you are on the right side of the boat then you can shout starboad which means they have to give way to you.
